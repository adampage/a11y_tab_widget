Keyboard Controls

When navigating through the page via keyboard controls, a tabbed interface works in the following ways:

Within the tablist

  • The selected tab in the tablist can receive focus, while other tabs are removed from the tabindex.

  • When focus is within the tablist the right & down arrows move focus to the next tab in the tablist, while left & up move focus to the previous tab in the tablist.

    If the first tab in the tablist is focused, and the left or up keys are pressed, then focus will move to the last tab in the tablist. The inverse will happen when the right or down keys are pressed while focused on the last tab in the tablist.

  • When focused on a tab, hitting Space Bar or Enter will activate the associated tab panel.

Within a tabpanel

When focused within a tab panel different keyboard controls become available.

  • CTRL + up moves focus to the currently selected tab.
  • CTRL + page up moves focus from the currently selected tab to previous tab in the tablist. In the case of the current tab being the first in the tablist order, then this action will move focus to the last tab in the tablist.
  • CTRL + page down will move focus from the currently selected tab to the next tab in the tablist. In the case of the current tab being the last in the tablist order, then this action will move focus to the first tab in the tablist.
  • Once focus has been moved from the panel, back to the tabs in the tablist, the tablist keyboard controls become active again.
