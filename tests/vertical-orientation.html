The default tabpanel

Not only is this Tab Widget nested within another, it also has data-atabs-orientation="vertical" set to the widget wrapper. This attribute with the "vertical" value will convert a tablist to a stacked UI. aria-orientation="vertical" will be set to the tablist, and up and down arrow keys, instead of left and right arrow keys will be used to navigate the individual tabs.

This tablist will be repositioned to the top of the Tab Widget container. And this message will be deleted when the script runs. So if you're seeing this, hi!
